# AIOE-Keyboard-Setup
Custom Keyboard with shortcut 
(look at README file and not preview)

Keyboard Layout:
1 2 3 4 5 6 7 8 9 0 x y =
u a t r f p j w z ' ( ) ^
 i o e c b v g q @ + /
  n k s . d h m l ; ?


Shifted Layer:
1 = _ 
3 = #
4 = $
5 = %
7 = &
0 = `
' = "
( = [
) = ]
. = ,
; = :
? = !
+ = -
/ = \
^ = <

Right Alt (Ctrl + Alt) Layer:
0 = ~
^ = >
+ = *
( = {
) = }
